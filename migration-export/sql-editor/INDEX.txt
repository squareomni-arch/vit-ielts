## Lưu ý quan trọng
- Mỗi file bắt đầu bằng: SET session_replication_role = replica;
  → Tắt FK trigger trong session, tránh lỗi FK trong lúc import
  → Tự bật lại ở cuối file: SET session_replication_role = DEFAULT;
- Nếu gặp lỗi "duplicate key": thêm ON CONFLICT DO NOTHING vào câu INSERT đó
- data_posts_002.sql (~3.6MB): nếu SQL Editor từ chối size, tách thêm bằng tay
- KHÔNG cần chạy các file data_ còn lại trong thư mục (là file gốc, đã được gộp/xử lý)
